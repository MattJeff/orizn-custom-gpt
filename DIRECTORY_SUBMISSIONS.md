**Long description**:
Orizn Visa API provides instant access to visa requirements for 39,585 passport-destination pairs (199 x 199 countries) in 15 languages. Data sourced from 136 official government sources, verified and updated regularly.

Available integrations:
- REST API with free tier (3,000 req/month)
- MCP Server (npm: orizn-visa-mcp) for Claude, Cursor, Codex, and any MCP client
- LangChain tools (Python: langchain-orizn, JS: @orizn/langchain)
- Custom GPT on the GPT Store

Visa types supported: visa_free, visa_required, e_visa, visa_on_arrival, eta, no_admission.
